Fly3D Frontends Guide

Fly3D SDK is a real-time 3D games and applications development tool. Therefore, it provides several development and production applications, forming the front-end layer of the SDK. This guide has the goal of introducing these applications, explaining and exemplifying their use and functionalities.

Below are the frontends covered in this guide:

Main Frontend: flyFrontend.exe
Parameter Editor: flyEditor.exe
Build Frontend: flyBuilder.exe
Shaders Editor: flyShader.exe
Multiplayer Server: flyServer.exe
ActiveX Control: fly3d.ocx
